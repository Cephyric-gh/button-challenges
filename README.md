# Button challenge preview

Web app to preview **Legends of IdleOn** World 7 **Button** challenges requirement values by completion step.

## Run locally

Requires [Node.js](https://nodejs.org/) (LTS recommended).

```bash
npm install
npm run dev
```

Open the URL Vite prints (usually `http://localhost:5173`).

## Scripts

| Command                | Description                        |
| ---------------------- | ---------------------------------- |
| `npm run dev`          | Start dev server with hot reload   |
| `npm run build`        | Production build to `dist/`        |
| `npm run preview`      | Serve the production build locally |
| `npm run lint`         | ESLint (strict + type-aware)       |
| `npm run lint:fix`     | ESLint with `--fix`                |
| `npm run format`       | Prettier write                     |
| `npm run format:check` | Prettier check (CI-friendly)       |

## Deployment
